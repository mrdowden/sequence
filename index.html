Sequence

Welcome to Sequence, the game of guessing number sequences.

Each number is a single digit, 0-9, with no number appearing more than once. You will have 10 chances to guess the correct number sequence.

To play, enter one number per field and click "Make Guess". You will be given feedback as follows:

  • Green means you have correctly placed a number in the sequence
  • Yellow means the number exists within the sequence, but in a different position
  • Red means the number does not exist anywhere in the sequence
